# Assignments

This directory contains all course assignments and homework.

## Structure

Each assignment should be organized in its own subdirectory with the following structure:

```
assignment-name/
├── README.md           # Assignment description and requirements
├── solution.py         # Your solution implementation
├── tests.py           # Test cases (if applicable)
└── data/              # Any required data files
```
